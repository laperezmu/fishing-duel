# Progresion de run

## Estado

- Estado: `pendiente`
- Prioridad: `alta`
- Depende de: `01_foundations/04_fail_success_conditions.md`, `03_content/01_zones.md`

## Objetivo

Describir como escala una run de principio a fin y que curvas de dificultad y recompensa debe sentir el jugador.

## Debe incluir

- inicio debil
- crecimiento de build
- salto de riesgo por zona
- preparacion para la zona legendaria

## Criterio de cierre

- La run tiene un arco reconocible.
- Se entiende como el jugador pasa de supervivencia a caza de trofeos.
