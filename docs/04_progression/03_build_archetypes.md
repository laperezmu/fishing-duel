# Arquetipos de build

## Estado

- Estado: `pendiente`
- Prioridad: `media`
- Depende de: `03_content/04_items_and_sponsorships.md`

## Objetivo

Definir que tipos de builds quiere soportar el MVP para guiar tienda, objetos y balance.

## Builds sugeridas

- control de distancia
- seguridad de profundidad
- rush de cuota
- caza de legendarios
- valor o reputacion

## Criterio de cierre

- Existen de 3 a 5 arquetipos claros.
- Cada arquetipo tiene al menos una ruta de objetos coherente.
