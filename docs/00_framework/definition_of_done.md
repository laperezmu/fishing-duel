# Definition of Done de diseno

Un tema solo puede pasar a `cerrado` si cumple todo esto:

- Tiene un objetivo de diseno claro.
- Define la decision que toma el jugador.
- Explica el efecto sobre la fantasia y el ritmo de la run.
- Contiene reglas lo bastante precisas para no contradecir otras fichas.
- Declara que catalogos toca.
- Lista preguntas abiertas en `ninguna` o explicita por que no bloquean el MVP.
- Tiene criterio de cierre verificable.
- El backlog maestro esta actualizado.

## No cerrar un tema si...

- depende de una regla aun no resuelta
- usa palabras no definidas en el glosario
- mezcla deseos futuros con reglas del MVP
- necesita otro documento para entenderse

## Salida minima al terminar una sesion

- que quedo cerrado
- que quedo abierto
- que tema sigue
