# Catalogo de peces

## Estado

- Estado: `pendiente`
- Prioridad: `alta`
- Depende de: `01_zones`, `02_systems/04_fish_decks_and_affinities.md`

## Objetivo

Construir el roster del MVP con peces memorables y legibles.

## Cada pez debe incluir

- nombre
- rol o arquetipo
- zona de aparicion
- identidad de color
- identidad de profundidad
- nivel de dificultad
- valor de reputacion
- nota tematica

## Criterio de cierre

- El MVP tiene un roster suficiente para variedad temprana, media y final.
- Ningun pez es redundante con otro en comportamiento base.
