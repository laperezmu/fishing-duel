# Catalogo de habilidades del pez

## Estado

- Estado: `pendiente`
- Prioridad: `alta`
- Depende de: `02_fish_catalog`

## Objetivo

Listar las habilidades del pez del MVP en lenguaje de diseno, no tecnico.

## Cada habilidad debe incluir

- nombre
- fantasia
- efecto jugable
- tipo de presion que agrega
- que peces pueden usarla

## Criterio de cierre

- El conjunto es pequeno y legible.
- Cada habilidad aporta una tension distinta.
