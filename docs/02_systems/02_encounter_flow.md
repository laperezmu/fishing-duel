# Flujo del encuentro

## Estado

- Estado: `pendiente`
- Prioridad: `critica`
- Depende de: `01_casting`

## Objetivo

Documentar el orden exacto de un combate desde el pique hasta captura automatica o escape.

## Debe incluir

- inicio del encuentro
- eleccion de color del jugador
- revelacion de carta del pez
- resolucion de color o habilidad
- movimiento de tracks
- descarte
- chequeos de captura, escape y reshuffle

## Criterio de cierre

- La secuencia no tiene huecos.
- Cualquier combate puede resolverse leyendo solo este archivo y las fichas que referencia.
