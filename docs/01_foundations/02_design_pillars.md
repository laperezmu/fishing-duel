# Pilares de diseno

## Estado

- Estado: `en_discusion`
- Prioridad: `critica`

## Pilares base

- Claridad: el jugador entiende por que gana o pierde.
- Riesgo y recompensa: el lance lejano acelera pero complica.
- Lectura: el mazo del pez genera informacion parcial valiosa.
- Pesca deportiva: reputacion, patrocinio y trofeos por encima de consumo.
- Rejugabilidad: cada run cambia por builds, pools y ruta.

## Falta cerrar

- priorizacion final de pilares
- que pillar se sacrifica si dos entran en conflicto

## Criterio de cierre

- La lista final no supera 5 pilares.
- Cada pilar tiene una explicacion de 1 a 3 lineas.
- Cada pilar tiene al menos una implicacion concreta sobre el diseno.
