# Condiciones de progreso y fracaso

## Estado

- Estado: `pendiente`
- Prioridad: `critica`
- Depende de: `03_run_loop`

## Objetivo

Definir cuando una run avanza, cuando termina y que se conserva.

## Preguntas a resolver

- que es una run ganada en el MVP
- que es una run perdida
- que pasa si se acaba el tiempo antes de llegar al legendario
- que recompensas persisten entre runs

## Criterio de cierre

- Existe una definicion clara de victoria.
- Existe una definicion clara de derrota.
- Existe una lista de recompensas de run y meta.
