# Exclusiones del MVP

## Estado

- Estado: `pendiente`
- Prioridad: `media`

## Objetivo

Dejar explicito que ideas se aparcan para evitar crecimiento descontrolado.

## Candidatos iniciales

- mazo o mano del jugador
- clima complejo
- eventos aleatorios extensos
- rotura persistente de equipo
- demasiadas habilidades avanzadas del pez
- minijuegos extra de cobro final

## Criterio de cierre

- Existe una lista de fuera de alcance para el MVP.
- Toda idea nueva que no quepa entra aqui o en parking lot.
