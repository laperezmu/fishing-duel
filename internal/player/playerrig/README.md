# internal/player/playerrig

Define las capacidades operativas de las herramientas del jugador.

## Responsabilidad

- Describir alcance horizontal maximo del jugador.
- Describir profundidad maxima alcanzable por el jugador.
- Servir como base configurable para futuras mejoras del loadout.

## Regla de arquitectura

- Este paquete modela las capacidades del jugador, no el comportamiento del pez.
- Sus valores pueden modificarse en futuras iteraciones por items o build.
